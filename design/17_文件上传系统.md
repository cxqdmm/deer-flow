# 文件上传与转换（Uploads）

---

## 1. 概述

DeerFlow 的文件上传系统处理用户上传的文档、图片等文件，转换为 Agent 可读的格式，并注入到对话上下文中。

```
用户上传文件
    │
    ▼
POST /api/threads/{thread_id}/uploads
    │
    ▼
Gateway 存储文件
    │
    ├─ PDF/PPT/Excel/Word → markitdown 转换
    └─ 图片/其他 → 保留原格式
    │
    ▼
UploadsMiddleware 注入文件列表
    │
    ▼
Agent 通过 /mnt/user-data/uploads/ 访问
```

---

## 2. 上传 API

```python
# app/gateway/routers/uploads.py

@router.post("/threads/{thread_id}/uploads")
async def upload_files(thread_id: str, files: list[UploadFile]):
    """上传文件到线程"""
    # 1. 验证：拒绝目录路径
    # 2. 存储到 threads/{thread_id}/user-data/uploads/
    # 3. 文档转换
    # 4. 记录元数据到 state.uploaded_files
    return {
        "success": True,
        "files": [
            {
                "filename": "report.pdf",
                "path": "uploads/report.pdf",
                "virtual_path": "/mnt/user-data/uploads/report.pdf",
                "artifact_url": "/api/threads/.../artifacts/...",
                "converted": True,  # 是否已转为 Markdown
                "size": 12345,
            }
        ]
    }
```

---

## 3. 支持格式

| 格式 | 转换方式 | 输出 |
|------|---------|------|
| PDF | markitdown | Markdown |
| PPT / PPTX | markitdown | Markdown（保留幻灯片结构） |
| Excel（.xlsx/.xls） | markitdown | Markdown 表格 |
| Word（.docx/.doc） | markitdown | Markdown |
| 图片（jpg/png/webp） | 保留 | 原文件（供 view_image） |
| 其他 | 保留 | 原文件 |

## 4. UploadsMiddleware

```python
# packages/harness/deerflow/agents/middlewares/uploads_middleware.py

class UploadsMiddleware:
    def after_model(self, output, messages, state, config):
        uploaded_files = state.get("uploaded_files", [])
        if not uploaded_files:
            return

        # 格式化为文件列表字符串
        file_list = "\n".join([
            f"- {f['filename']} ({f['size']} bytes)"
            for f in uploaded_files
        ])

        message = f"Uploaded files:\n{file_list}"
        messages.append(HumanMessage(content=message))
```

注入后的消息示例：

```
Uploaded files:
- report.pdf (12,345 bytes) [converted to Markdown]
- data.xlsx (5,678 bytes) [converted to Markdown]
- image.png (89,012 bytes)
```

Agent 可以通过 `/mnt/user-data/uploads/report.pdf` 读取转换后的 Markdown 内容。

## 5. 文档转换

```python
# packages/harness/deerflow/uploads/converter.py

async def convert_document(file_path: Path, output_dir: Path) -> Path | None:
    """将文档转换为 Markdown"""
    import markitdown

    converter = markitdown.MarkItDown()
    result = converter.convert(str(file_path))

    output_path = output_dir / f"{file_path.stem}.md"
    output_path.write_text(result.text_content, encoding="utf-8")
    return output_path
```

## 6. 安全限制

### 6.1 路径验证

```python
# 拒绝目录路径上传
for file in files:
    if Path(file.filename).is_absolute():
```

## 7. 线程清理

当 LangGraph 线程被删除后，Gateway 负责清理本地文件：

```python
# app/gateway/routers/threads.py

@router.delete("/threads/{thread_id}")
async def delete_thread(thread_id: str):
    # 删除 .deer-flow/threads/{thread_id}/
    Paths.delete_thread_dir(thread_id)
```
