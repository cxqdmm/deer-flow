## 关键数据流

### Agent 请求流程
```
Client → Nginx (2026)
  ├─ /api/langgraph/* → LangGraph Server (2024)
  │   ├─ ThreadDataMiddleware: 创建线程目录
  │   ├─ UploadsMiddleware: 注入上传文件
  │   ├─ SandboxMiddleware: 获取沙箱环境
  │   ├─ DanglingToolCallMiddleware: 补全悬空工具调用
  │   ├─ GuardrailMiddleware: 权限检查（可选）
  │   ├─ SummarizationMiddleware: 上下文压缩（可选）
  │   ├─ TodoListMiddleware: 任务跟踪（可选，plan_mode）
  │   ├─ TitleMiddleware: 自动生成标题
  │   ├─ MemoryMiddleware: 记忆更新入队
  │   ├─ ViewImageMiddleware: 图像处理（vision 模型）
  │   ├─ SubagentLimitMiddleware: 子 Agent 并发限制
  │   └─ ClarificationMiddleware: 截获澄清请求
  └─ /api/* → Gateway API (8001)
      ├─ GET/PUT /api/models
      ├─ GET/PUT /api/mcp/config
      ├─ GET/PUT /api/skills
      ├─ POST /api/skills/install
      ├─ GET /api/memory
      ├─ POST /api/threads/{id}/uploads
      └─ GET /api/threads/{id}/artifacts/{path}
```

### 沙箱虚拟路径映射
| 虚拟路径 | 物理路径 |
|---------|---------|
| `/mnt/user-data/workspace` | `backend/.deer-flow/threads/{thread_id}/user-data/workspace` |
| `/mnt/user-data/uploads` | `backend/.deer-flow/threads/{thread_id}/user-data/uploads` |
| `/mnt/user-data/outputs` | `backend/.deer-flow/threads/{thread_id}/user-data/outputs` |
| `/mnt/skills` | `deer-flow/skills/` |
| `/mnt/acp-workspace` | `{base_dir}/threads/{thread_id}/acp-workspace/` |

### 双运行时模式
```
标准模式（make dev）:
  nginx(2026) + frontend(3000) + gateway(8001) + langgraph(2024) = 4 进程

Gateway 模式（make dev-pro）:
  nginx(2026) + frontend(3000) + gateway(8001, 含 Agent Runtime) = 3 进程
```
