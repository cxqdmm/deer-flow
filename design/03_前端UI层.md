# 前端 UI 层（Next.js 16 + React 19）

---

## 1. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) |
| UI 库 | React 19 |
| 语言 | TypeScript 5.8 |
| 样式 | Tailwind CSS 4 |
| 组件库 | Shadcn UI, MagicUI, React Bits |
| AI 集成 | LangGraph SDK, Vercel AI Elements |
| 包管理 | pnpm 10.26.2 |
| 开发服务器 | Turbopack（热更新） |

---

## 2. 页面路由

```
/                               # Landing Page（首页）
/workspace                      # Workspace 布局
/workspace/chats               # 对话列表
/workspace/chats/new            # 新建对话
/workspace/chats/[thread_id]   # 特定对话页
/workspace/agents              # Agent 列表
/workspace/agents/[agent_name] # 特定 Agent
/workspace/agents/new          # 新建 Agent
```

---

## 3. 核心架构

```
用户操作
    │
    ▼
React 组件（"use client"）
    │
    ▼
Thread Hooks（useThreadStream, useSubmitThread, useThreads）
    │
    ▼
LangGraph SDK 流式客户端
    │
    ▼
Nginx → LangGraph Server (2024)
    │
    ▼
SSE 流式事件
    │
    ▼
Thread State 更新 → UI 渲染
```

## 4. 核心模块（core/）

### 4.1 threads（线程管理）

| 文件 | 说明 |
|------|------|
| `core/threads/hooks.ts` | 主要 Hook：`useThreadStream`, `useSubmitThread`, `useThreads` |
| `core/threads/types.ts` | `AgentThread`, `AgentThreadContext` 类型定义 |
| `core/threads/export.ts` | 线程导出功能 |

**ThreadContext**：

```typescript
interface AgentThreadContext {
  thread_id: string;
  model_name: string | undefined;
  thinking_enabled: boolean;
  is_plan_mode: boolean;
  subagent_enabled: boolean;
}
```

### 4.2 api（LangGraph 客户端）

```typescript
// core/api/index.ts
const client = getAPIClient();  // LangGraph SDK 单例
```

LangGraph SDK 提供：
- `client.threads.create()`
- `client.threads.stream()`
- `client.agents.get()`
- `client.runs.stream()`

### 4.3 其他模块

| 模块 | 说明 |
|------|------|
| `core/artifacts/` | Artifact 加载与缓存 |

## 6. 流式数据流

```typescript
// useThreadStream 的 SSE 事件处理
for await (const event of client.threads.stream(...)) {
  if (event.event === "messages/partial") {
    // 逐字更新 AI 响应
    updateMessage(event.data);
  }
  if (event.event === "metadata") {
    // 元数据更新（标题、artifacts、todos）
    updateMetadata(event.data);
  }
  if (event.type === "task_started") {
    // 子 Agent 事件
    showTaskIndicator(event.data);
  }
}
```

## 9. 环境变量

```bash
NEXT_PUBLIC_BACKEND_BASE_URL=http://localhost:8001  # 可选，默认通过 Nginx
NEXT_PUBLIC_LANGGRAPH_BASE_URL=http://localhost:2024  # 可选，默认通过 Nginx
```

前端默认通过 Nginx 代理访问后端，也可以直接指定 URL。
