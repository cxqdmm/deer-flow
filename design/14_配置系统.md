# 配置系统（config.yaml + extensions_config.json）

---

## 1. 概述

DeerFlow 使用两个配置文件：

| 文件 | 内容 | 位置 |
|------|------|------|
| `config.yaml` | 主配置（模型、工具、沙箱、记忆等） | 项目根目录 |
| `extensions_config.json` | MCP 服务器和技能状态 | 项目根目录 |

配置文件路径优先级：

```
1. 环境变量指定路径（DEER_FLOW_CONFIG_PATH）
2. 当前目录（backend/config.yaml）
3. 项目根目录（deer-flow/config.yaml）← 推荐
```

## 2. config.yaml 完整结构

```yaml
# ============================================================================
# 配置版本（自动升级用）
# ============================================================================
config_version: 5

# ============================================================================
# 日志
# ============================================================================
log_level: info                              # debug/info/warning/error

# ============================================================================
# Token 使用统计
# ============================================================================
token_usage:
  enabled: false

# ============================================================================
# 模型配置
# ============================================================================
models:
  - name: gpt-4
    display_name: GPT-4
    use: langchain_openai:ChatOpenAI         # LangChain 类路径
    model: gpt-4
    api_key: $OPENAI_API_KEY
    base_url: https://api.openai.com/v1       # 可选
    request_timeout: 600.0
    max_retries: 2
    max_tokens: 4096
    temperature: 0.7
    supports_thinking: false
    supports_vision: true
    # Thinking 模型专用：
    when_thinking_enabled:
      extra_body:
        thinking:
          type: enabled

# ============================================================================
# 工具分组
# ============================================================================
tool_groups:
  - name: search
    tools: ["web_search"]
  - name: web
    tools: ["web_fetch"]
  - name: file
    tools: ["ls", "read_file", "write_file", "glob", "grep", "str_replace"]
  - name: bash
    tools: ["bash"]

# ============================================================================
# 工具定义
# ============================================================================
tools:
  - name: web_search
    use: deerflow.community.ddg_search.tools:web_search_tool
    group: search
    tool_extra:
      max_results: 5

# ============================================================================
# 工具搜索（延迟加载）
# ============================================================================
tool_search:
  enabled: false

# ============================================================================
# ACP Agents
# ============================================================================
acp_agents: {}

# ============================================================================
# 沙箱配置
# ============================================================================
sandbox:
  use: deerflow.sandbox.local:LocalSandboxProvider
  allow_host_bash: false
  mounts: []
  bash_output_max_chars: 20000
  read_file_output_max_chars: 50000
  ls_output_max_chars: 20000

# ============================================================================
# 技能配置
# ============================================================================
skills:
  path: ./skills
  container_path: /mnt/skills

# ============================================================================
# Skill Evolution（实验性）
# ============================================================================
skill_evolution:
  enabled: false

# ============================================================================
# 自动标题生成
# ============================================================================
title:
  enabled: true
  max_words: 10
  max_chars: 80
  prompt_template: |

# ============================================================================
# 上下文摘要
# ============================================================================
summarization:
  enabled: false
  trigger:
    type: tokens
    value: 60000
  keep:
    type: last_n_messages
    value: 10

# ============================================================================
# 记忆系统
# ============================================================================
memory:
  enabled: true
  injection_enabled: true
  storage_path: .deer-flow/memory.json
  debounce_seconds: 30
  model_name: null
  max_facts: 100
  fact_confidence_threshold: 0.7
  max_injection_tokens: 2000

# ============================================================================
# 子 Agent
# ============================================================================
subagents:
  enabled: true

# ============================================================================
# IM 频道
# ============================================================================
channels:
  langgraph_url: http://localhost:2024
  gateway_url: http://localhost:8001

# ============================================================================
# 权限检查（Guardrails）
# ============================================================================
guardrails:
  enabled: false
```

## 3. extensions_config.json

```json
{
  "mcpServers": {
    "github": {
      "enabled": true,
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "$GITHUB_TOKEN"
      }
    }
  },
  "skills": {
    "deep-research": {
      "enabled": true
    }
  }
}
```

## 4. 配置版本管理

```yaml
# config.example.yaml
config_version: 5
```

当用户 `config.yaml` 版本低于示例时：

```
WARNING - Your config.yaml (version 3) is outdated — the latest version is 5.
Run `make config-upgrade` to merge new fields.
```

执行 `make config-upgrade` 自动合并新字段（保留用户现有值，创建 `.bak` 备份）。

## 5. 环境变量解析

配置值中 `$VAR_NAME` 自动解析为环境变量：

```yaml
api_key: $OPENAI_API_KEY          # → 从环境变量读取
env:
  GITHUB_TOKEN: $GITHUB_TOKEN     # → 从环境变量读取
```

## 6. 配置类层次

```
AppConfig
├── log_level: str
├── token_usage: TokenUsageConfig
├── models: list[ModelConfig]
├── sandbox: SandboxConfig
├── tools: list[ToolConfig]
├── tool_groups: list[ToolGroupConfig]
├── skills: SkillsConfig
├── skill_evolution: SkillEvolutionConfig
├── extensions: ExtensionsConfig          # → extensions_config.json
├── tool_search: ToolSearchConfig
├── title: TitleConfig
├── summarization: SummarizationConfig
├── memory: MemoryConfig
├── subagents: SubagentsAppConfig
├── guardrails: GuardrailsConfig
├── checkpointer: CheckpointerConfig
└── stream_bridge: StreamBridgeConfig
```
